🧠 MindCare Therapy Booking System

A full-stack web application that allows users to book therapy appointments online while providing a secure admin dashboard to manage appointments, view analytics, and export reports.

📖 Project Overview

MindCare Therapy Booking System is designed to simplify the appointment booking process for patients seeking mental health support. Users can easily schedule therapy sessions, while administrators can securely manage appointments, monitor booking statistics, and generate reports through an interactive dashboard.
